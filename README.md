# 🤖 AI Chatbot using Hugging Face API

🔗 **Live Demo:** [Click here to try the chatbot]([https://your-streamlit-url-here.streamlit.app](https://ai-chatboat-project-d8jyflawzjhyzcp4yitsdf.streamlit.app/))


A simple conversational AI chatbot built with Streamlit and Hugging Face's Inference API, featuring prompt engineering techniques like custom system prompts and few-shot examples.

## Features
- Real-time chat interface built with Streamlit
- Powered by an open-source LLM (Kimi-K2-Instruct) via Hugging Face Inference API
- Custom system prompt defining bot personality and behavior
- Few-shot examples for consistent response style
- Handles limitations gracefully (e.g., no real-time data access)

## Tech Stack
- Python
- Streamlit
- Hugging Face Inference API (OpenAI-compatible)
- python-dotenv
